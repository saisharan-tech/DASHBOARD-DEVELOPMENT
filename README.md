## PROJECT DESCRIPTION

The Interactive Sales Analysis Dashboard is a data visualization project developed using Power BI to analyze and monitor sales performance. The main objective of this project is to transform raw sales data into meaningful insights through interactive dashboards and visualizations.

This dashboard provides a comprehensive view of key business metrics such as total revenue, units sold, sales trends over time, and performance across different products, employees, and supervisors. It helps stakeholders quickly understand sales patterns and make data-driven decisions.

The project uses a structured dataset containing sales details such as product ID, employee name, supervisor, date, revenue, and units sold. By leveraging Power BI's capabilities, the data is cleaned, modeled, and visualized in an interactive and user-friendly format.

---

## FEATURES

- Interactive filters (Date, Product ID, Employee, Supervisor, Product Name)  
- KPI cards displaying Total Revenue and Units Sold  
- Time-series analysis of revenue trends  
- Product-wise revenue and unit analysis  
- Supervisor-wise revenue distribution (Pie Chart)  
- Employee-wise performance visualization (Treemap)  
- Clean and visually appealing dashboard design  

---

## TOOLS & TECHNOLOGIES USED

- Power BI (Data visualization and dashboard creation)  
- Microsoft Excel (Data source)  
- DAX (Data Analysis Expressions for calculations)

## DASHBOARD INSIGHTS

The dashboard provides several valuable insights, including:

- Identification of top-performing products and employees  
- Revenue trends across different time periods  
- Supervisor contribution to total revenue  
- Comparison between revenue and units sold  
- Detection of sales patterns and performance gaps  

---

## WORKING

The project begins with importing the sales dataset into Power BI. Data cleaning and transformation are performed to ensure consistency and accuracy. Relationships between different data fields are established for proper analysis.

DAX measures are created to calculate key metrics such as total revenue and total units sold. Various visualizations are then built, including bar charts, line charts, pie charts, and treemaps.

Interactive slicers are added to allow users to filter data dynamically based on different parameters like date, product, and employee. This enhances the usability and flexibility of the dashboard.
